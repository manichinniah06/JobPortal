# 💼 Job Portal (Frontend)

A responsive Job Portal web application UI that allows users to browse job listings and simulate job application flows.

⚠️ Note: This is a frontend-only project (no backend integration).

---

## 🚀 Features

- Browse job listings
- View job details
- Apply button (UI only)
- Responsive design

---

## ⚙️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 📂 Project Structure

- index.html
- styles.css
- script.js

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/manichinniah06/JobPortal.git
```

2. Open `index.html` in browser
